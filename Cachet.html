Hackathon - Cachet

Worked with an ALM (application lifestyle management) team to automate AWS Console monitors which posted AWS app downtime/recovery to Cachet status pages.

For each newly deployed AWS app, a new instance of Cachet is spun up in a docker container on AWS. Health metrics are constantly pulled from AWS Console and pushed into an AWS database. These metrics are then fed into the status page to build the dashboard.

The solution was developed using Java, SQL, Databases, Docker, and Cachet API. Not able to publish source code, as it is sensitive information owned by CDK Global.
