Rapid7 Nexpose

Rapid7 Nexpose is a on-prem vulnerability management software. It allows you to define IP ranges to be port scanned for vulnerabilities.

Studied the Nexpose API and presented it to a team of security engineers. Created a script to define IP ranges and engines in Nexpose from a csv file. Developed a program to generate a vulnerability report on a single IP address in csv format.

The solutions were developed using Python and Nexpose API.
